# -numpy-
用numpy实现了一个很简单的神经网络，注释比代码写的还要多。。希望对初学者会有用。如果有用的话不要忘了star一下~


根据kaggle上的mnist比赛，我写了这段代码，并且注释了相关的知识点和链接。

代码主要包括：
   1.数据的加载和预处理
   2.神经网络的搭建
   3.用训练好的神经网络产生预测值，并储存成kaggle要求的格式

1.数据的加载和预处理：
   这一块主要包括如何读取和处理csv文件，如何制作softmax标签，如何将像
   素值转化为矩阵，并且讲解了如何使用pickle来加快数据的载入速度。

 2.神经网络的搭建：
   这一部分应该算重头戏了，在这一块主要包含了参数的初始化，正向传播，
   loss和准确率的计算，反向传播和梯度下降。需要注意的是，虽然有很多成熟的
   深度学习框架，比如Tensorflow，Keras, pytorch等。但是！我们不能只做一个
   调参侠和调包师。因此，我通过numpy，用最原始的矩阵运算实现了这个简单的
   神经网络，这样你可以直观地理解它的数学原理。

3. 预测：
   我们所做的所有工作就是为了让模型能够在测试集上跑出好的结果。因此，神经
   网络训练完后，我们需要在测试集上跑一遍（其实和在验证集上与训练集上跑没
   什么不同，只是少了loss和准确率的计算，自然也少了反向传播与梯度下降。），
   然后把结果保存成kaggle要求我们的格式（详见sample_submission.csv的格式）
